# Exmd

print github's README.md on terminal.

## install

install [elixir](https://github.com/elixir-lang/elixir)

```
$ git clone chitacan/exmd
$ cd exmd
$ mix deps.get
$ mix escript.install
```

## run

```
$ exmd elixir-lang/elixir
```
